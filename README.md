# 100Hires Portfolio Project - Stage 1

## Tools Installed
* **Cursor IDE**: Version 0.x (Latest)
* **Claude Code for VS Code** (Anthropic Verified Extension)
* **Codex** (AI Assistant Extension)

## Steps Completed
1. Downloaded and configured Cursor IDE as the primary development environment.
2. Signed in and linked Cursor with my personal GitHub account.
3. Created a new public GitHub repository named `100hires`.
4. Cloned the repository locally using the integrated terminal in Cursor.
5. Installed and authenticated both the `Claude Code` and `Codex` extensions inside Cursor.
6. Documented the setup process and troubleshooting steps in this README.md file.

## Issues Encountered & Solutions
* **Issue 1: Extension Shortcut Inconsistency** * *Problem*: The default shortcut `Ctrl + Shift + X` didn't immediately open the Extensions tab.  
  * *Solution*: I navigated manually using the Extension icon on the left sidebar to locate the marketplace.
* **Issue 2: Multiple Extension Variants** * *Problem*: Searching for "Claude Code" showed various third-party chat history and log viewers.  
  * *Solution*: I carefully analyzed the publisher names and ensured I installed the official, verified **"Claude Code for VS Code" by Anthropic** to maintain security and tool compliance.
